# Portland Film Festival

A modern, cinematic website for the [Portland Film Festival](https://portlandfilm.org) — Portland's leading indie film non-profit supporting independent voices since 2013.

## Tech Stack

- **React 18** with React Router v6
- **Vite** for blazing fast development and builds
- **Tailwind CSS** for utility-first styling
- **Lucide React** for icons
- **Framer Motion** (available for animation enhancements)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment (Vercel)

1. Push to GitHub
2. Import repo in [Vercel](https://vercel.com)
3. Vercel auto-detects Vite — no configuration needed
4. Deploy!

Or use the Vercel CLI:

```bash
npx vercel
```

| Route | Page |

| `/` | Home — Hero, about, categories, news, CTAs |
| `/about` | Mission, goals, values |
| `/submit` | Film submission info, deadlines, platforms |
| `/volunteer` | Volunteer opportunities and benefits |
| `/sponsor` | Sponsorship information |
| `/contact` | Contact form and festival details |
| `/news` | Festival news and press coverage |

## Design

The design follows a **cinematic noir** aesthetic with:

- Deep dark backgrounds (`#06060a`)
- Gold/amber accent color (`#f4c430`) evoking theater marquee lights
- Playfair Display for headlines, DM Sans for body text
- Film grain overlay effect
- Sprocket-hole dividers and film strip borders
- Subtle glow and spotlight animations
- Scroll-reveal animations
